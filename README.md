# 🎯 C Number Guessing Game

A simple number guessing game written in C.

## 📌 Description
This program generates a random number between 1 and 100 , and the player has to guess it.

After each guess, the program gives hints:
- 📉 Too low
- 📈 Too high
- ✅ Correct guess

## 🛠️ Technologies Used
- C Programming Language
- Standard Libraries (stdio.h, stdlib.h, time.h)

## ▶️ How to Run

1. Compile the program:
  gcc number_guessing_game.c -o game
2. Run the program :
   ./game

## 💡 Features
- Random number generation
- Unlimited attempts
- Hint system (high/low)
- Tracks number of attempts
